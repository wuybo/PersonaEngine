# 分身引擎
分身引擎，一机千面，自动掘金，浏览器任务多开管理

## 浏览器多开管理，可以对Google Edge 进行浏览器多端口 多开 管理；
支持 Google Edge 等主流浏览器的多实例、多端口、独立环境运行。

每个分身拥有独立 Cookie、缓存与 IP 配置，完美隔离账号，适用于运营、测试、爬虫等场景。

通过 JavaScript / Python 脚本，对指定端口的浏览器实例进行精细化操作。

无论是自动填表、数据采集还是交互模拟，一切尽在掌控。
